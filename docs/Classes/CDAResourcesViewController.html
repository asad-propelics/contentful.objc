CDAResourcesViewController

@interface CDAResourcesViewController : UITableViewController

CDAResourcesViewController is designed to make it easy to display a list of Resources in a table view.

It will automatically request the data from a Space you specify with a query you define once it is visible on screen. Additionally, it provides a mapping for selecting which Fields to display in the cells of the table view.

Initializing the CDAResourcesViewController Object */
  • -initWithCellMapping:

    Initializes a new instance with the given cell mapping. The cell mapping is a dictionary, containing keypaths to properties on each cell and mapping them to Field identifiers. This is used for automatically filling new cells with data from the Resources.

    Example:

    @{ @"textLabel.text": @"fields.name" }

    This will set the text of each cell’s textLabel to the corresponding Entry’s value of Field name.

    Return Value

    An initialized CDAResourcesViewController or nil if the object couldn’t be created.

  • -initWithCellMapping:items:

    Initializes a new instance with the given cell mapping and a local array of Resources. This allows displaying a list of Resources which were already fetched from the server. In this case, the client and query properties will be completely ignored.

    initWithCellMapping: for a description of the cell mapping format.
