CDAEntriesViewControllerDelegate

@protocol CDAEntriesViewControllerDelegate <NSObject>

The delegate of a CDAEntriesViewController object must adopt the CDAEntriesViewControllerDelegate protocol. The one optional method of the protocol allows reacting to row selection by the user.

By default, a CDAFieldsViewController will be pushed to the entries view controller’ navigationController with the selected Entry as parameter.

  • -entriesViewController:didSelectRowWithEntry:

    This delegate method is called when the user selects a row in the table view of entries.

    Declaration

    Objective-C

    - (void)entriesViewController:
            (CDAEntriesViewController *_Nonnull)entriesViewController
        didSelectRowWithEntry:(CDAEntry *_Nonnull)entry;

    Swift

    optional func entriesViewController(_ entriesViewController: Any!, didSelectRowWith entry: CDAEntry)

    Parameters

    entriesViewController

    The sender of the delegate method.
    entry

    The Entry associated with the row the user selected.
