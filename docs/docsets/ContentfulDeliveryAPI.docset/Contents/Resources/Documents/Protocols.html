  • CDAEntriesViewControllerDelegate

    The delegate of a CDAEntriesViewController object must adopt the CDAEntriesViewControllerDelegate protocol. The one optional method of the protocol allows reacting to row selection by the user.

    By default, a CDAFieldsViewController will be pushed to the entries view controller’ navigationController with the selected Entry as parameter.
