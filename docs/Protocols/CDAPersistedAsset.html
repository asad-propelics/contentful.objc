CDAPersistedAsset

@protocol CDAPersistedAsset <NSObject>

Any class representing Assets saved to a persistent store needs to conform to this protocol.

If any of the optional properties are implemented, they will automatically be mapped to the corresponding Asset fields from Contentful.
